


* PACE package for Functional Data Analysis and Empirical Dynamics 
# [![](https://api.travis-ci.org/hadjipantelis/tPACE.svg?branch=master)](https://travis-ci.org/hadjipantelis/tPACE)

** Installation
You can install the package in R using:
#+BEGIN_SRC R
devtools::install_github("hadjipantelis/tPACE", ref='demo')
#+END_SRC

** Load Package and Data
Once installed you can load the package with:
#+BEGIN_SRC R
library(fdapace)
#+END_SRC
